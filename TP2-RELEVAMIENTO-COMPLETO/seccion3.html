PASO A PASO

PLANIFICACION Y ALCANCE

Se definen los límites del proyecto: qué sedes se visitarán, qué tipos de dispositivos se incluirán y cuáles se excluirán. Se asignan responsabilidades, se establecen cronogramas y se eligen las herramientas de software para el escaneo automático.

RECOPILACION DE INFORMACION

Fase de recolección de datos utilizando métodos mixtos. Se ejecutan herramientas automatizadas de descubrimiento de red (como agentes de software o escaneos IP) complementadas con inspecciones visuales en las áreas de trabajo para registrar números de serie físicos.

VERIFICACION Y REGISTRO

Se cruza la información obtenida automáticamente con los registros físicos. Se eliminan duplicados, se corrigen discrepancias (por ejemplo, números de serie mal leídos) y se consolidan los datos en una base de datos centralizada o CMDB.

ETIQUETADO Y CONTROL

Se colocan etiquetas físicas de identificación (códigos de barra, códigos QR o etiquetas RFID) a cada equipo verificado. Esto vincula el objeto físico con su registro digital, permitiendo auditorías rápidas con un lector de mano en el futuro

DOCUMENTACION

Se genera el informe técnico definitivo y el cuadro de mando. Se entregan los manuales de procedimientos para mantener el inventario actualizado, asegurando que cada alta, baja o modificación futura de hardware quede registrada bajo el mismo estándar.
